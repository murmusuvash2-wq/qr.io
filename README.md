# A2ZQR (EzQR.IO)

Bridge the physical and digital, instantly. A2ZQR is a production-grade QR Code Generation Platform focused on premium QR experiences, templates, and 100+ specialized tools.

## Features
- **100+ QR Tools:** Specialized generators for WiFi, Payments, Healthcare, Menus, Events, and more.
- **Lightning Fast:** Generates QRs on the client-side within milliseconds.
- **Premium Templates:** AI-powered and handcrafted design themes, frames, and background assets.
- **Dynamic Routing:** Change destination URLs dynamically without reprinting (requires Pro account).
- **Offline/Print Safe:** Focuses on high contrast and accurate scannability.

## Local Setup
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Configure your `.env` and `firebase-applet-config.json` (see `.env.example`).
4. Run `npm run dev` to start the local development server on port 3000.
5. Access the app at `http://localhost:3000`.

## Scripts
- `npm run dev`: Starts the development server.
- `npm run build`: Compiles for production.
- `npm run lint`: Runs ESLint over the `src` directory.
- `npm run format`: Formats code with Prettier.
